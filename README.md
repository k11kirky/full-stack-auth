 ## Installation

Clone the project

Go into the client dir and run yarn

 ### `cd client && yarn`

To start the app use

### `yarn start`

From root do the same for the server

 ### `cd ../server && yarn`

To start the app use

### `yarn start`

go to (http://localhost:3000)

the server is as (http://localhost:4000)
